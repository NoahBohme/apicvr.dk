MCP Server

CVR til din AI-assistent

Giv Claude, ChatGPT og andre AI-assistenter direkte adgang til Det Centrale Virksomhedsregister via Model Context Protocol (MCP).

Server URL

https://mcp.apicvr.dk/mcp

Hvad er MCP?

Model Context Protocol (MCP) er en åben standard der gør det muligt for AI-assistenter at forbinde til eksterne datakilder og tjenester i realtid. I stedet for at AI'en kun kender til data fra sin træning, kan den aktivt hente opdaterede oplysninger.

Med CVR MCP-serveren kan du spørge din AI-assistent om danske virksomheder — og den vil slå det op i CVR-registret, som om den selv brugte API'et. Perfekt til research, due diligence, leadgenerering og meget mere.

Tilgængelige funktioner

CVR MCP-serveren eksponerer 7 funktioner:

lookup_company
(cvr_number: int)
Hent alle oplysninger om en virksomhed via CVR-nummer
search_company_by_name
(name: str)
Søg virksomheder på navn (præfiks-match)
search_company_fuzzy
(name: str)
Fuzzy-søgning på virksomhedsnavn — finder resultater selvom stavningen ikke er helt præcis
search_company_by_email
(email: str)
Find virksomheder registreret med en given emailadresse
search_company_by_email_domain
(domain: str)
Find alle virksomheder med email på et bestemt domæne, f.eks. lego.com
search_company_by_phone
(phone: str)
Find virksomheder registreret med et bestemt telefonnummer
search_company_by_address
(address: str, postal_code?: str)
Find virksomheder på en bestemt adresse, valgfrit filtreret på postnummer

Opsætningsvejledning

Vælg din AI-assistent og følg vejledningen:

1
Claude.ai (web)
▶

Claude.ai understøtter MCP-integrationer fra Pro og Team planer. Tilføj CVR-serveren på under et minut:

  1. Gå til claude.ai/settings
  2. Klik på Integrations
  3. Klik på Add more eller Add integration
  4. Indsæt URL: https://mcp.apicvr.dk/mcp
  5. Klik Save — CVR-funktionerne er nu tilgængelige i alle dine samtaler
2
Claude Desktop (Mac / Windows)
▶

Åbn Claude Desktop's konfigurationsfil og tilføj CVR MCP-serveren:

Placering af konfigurationsfil:
Mac: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json

Tilføj følgende til filen (opret filen, hvis den ikke eksisterer):

{ "mcpServers": { "apicvr": { "command": "npx", "args": ["-y", "mcp-remote", "https://mcp.apicvr.dk/mcp"] } } }

Genstart Claude Desktop — du kan nu spørge om danske virksomheder direkte i dine chats.

3
Claude Code (CLI)
▶

Tilføj CVR MCP-serveren til Claude Code med én kommando:

claude mcp add apicvr --transport http https://mcp.apicvr.dk/mcp

Eller tilføj manuelt til .claude/settings.json i dit projekt:

{ "mcpServers": { "apicvr": { "type": "http", "url": "https://mcp.apicvr.dk/mcp" } } }
4
Andre MCP-kompatible klienter
▶

CVR MCP-serveren er fuldt kompatibel med alle klienter der understøtter MCP HTTP-transport (Streamable HTTP). Brug server-URL'en:

https://mcp.apicvr.dk/mcp

Serveren kræver ingen API-nøgle eller autentificering.
